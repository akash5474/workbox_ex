Example project demonstrating issue with [GoogleChrome/Workbox](https://github.com/googlechrome/workbox) v3.0.0-alpha.6

### Run Project

```
git clone -b <branch> https://github.com/akash5474/workbox_ex.git

npm install
npm run build
```
